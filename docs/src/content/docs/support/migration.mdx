This page tracks breaking changes between major versions of `poli-page/sdk`. For the full per-release diff, see the [changelog](../changelog/).

## v1.0

The first stable release. The full v1.0 surface and the renames from the pre-1.0 line are documented in [`MIGRATION.md` in the repo](https://github.com/poli-page/sdk-php/blob/main/MIGRATION.md).

Short version: every render call now goes through a namespace property (`$client->render->pdf()`, `$client->render->preview()`, `$client->render->document()`), all PDF-producing methods require `ProjectModeInput`, and inline mode is supported only on `render->preview` via `InlineModeInput`.
