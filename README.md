# poli-page/sdk (PHP SDK)

Official PHP SDK for [Poli Page](https://poli.page) — generate PDFs from any PHP application, with PSR-18 / PSR-17 HTTP transport so you can plug in Guzzle, Symfony HTTP Client, or your own.

> **Status**: scaffold only. Implementation begins in P2.0 of the [SDK roadmap](https://github.com/poli-page/poli-page/blob/develop/docs/onboarding/micka/sdk-roadmap.md).

## Install

```bash
composer require poli-page/sdk
```

## Recipes

Framework-agnostic recipes (WordPress, WooCommerce, Magento, Slim, plain PHP-FPM) live under `examples/` once implemented. First-party integrations for Laravel and Symfony live in their own repositories: [`poli-page/laravel`](https://github.com/poli-page/laravel) and [`poli-page/symfony-bundle`](https://github.com/poli-page/symfony-bundle).
